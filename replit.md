# digibeg Landing Page

## Overview
A German-language landing page for "digibeg" - a digital assistant application. This is a static website with no backend.

## Project Structure
- `index.html` - Main landing page with inline styles
- `style.css` - Additional CSS styles (not currently linked)
- `app.js` - JavaScript for smooth scrolling functionality

## Tech Stack
- Static HTML/CSS/JS
- Served with `serve` npm package

## Running Locally
The website is served using `npx serve -l 5000` on port 5000.

## Deployment
Configured for static deployment serving files from the root directory.
